## 注意事项

1. 每道题都要写清楚解题思路
2. 代码要有适当的注释
3. 考虑时间复杂度和空间复杂度
4. 测试边界情况和特殊输入
5. 定期复习已做过的题目

## 贡献指南

1. 按照题目类型分类存放代码
2. 每个题目包含：题目描述、解题思路、代码实现、复杂度分析
3. 代码风格保持一致
4. 及时更新刷题记录表格
